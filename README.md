# Full Stack Blog Application

## Overview

A full stack blog application built using React.js, Node.js, Express.js, MongoDB, JWT Authentication, and CRUD operations.

Users can:

* Register
* Login
* Create blog posts
* View blog posts
* Update blog posts
* Delete blog posts
* Logout securely

## Features

* User Authentication using JWT
* Password Hashing using bcrypt
* Create, Read, Update, Delete (CRUD) operations
* MongoDB Database Integration
* React Frontend
* Express Backend
* Responsive UI
* Protected Routes

## Tech Stack

### Frontend

* React.js
* Axios
* React Router DOM
* CSS

### Backend

* Node.js
* Express.js
* JWT
* bcryptjs

### Database

* MongoDB Atlas

## Installation

### Clone Repository

```bash
git clone https://github.com/Hema-01-hub/full-stack-blog-app.git
```

### Install Dependencies

Frontend:

```bash
cd client
npm install
```

Backend:

```bash
cd server
npm install
```

### Run Application

Backend:

```bash
node main.js
```

Frontend:

```bash
npm start
```
